# Figma to Code - Static Web Page

A responsive and static web page built to practice precisely translating modern UI layouts from Figma into clean, semantic HTML5 and CSS3 code.

## 🚀 Purpose & Learning Outcomes
* **Design Accuracy:** Focused on replicating the exact visual hierarchy, spacing, and typography from the Figma file.
* **File Structure:** Established a clean project architecture with organized asset folders.
* **CSS Layouts:** Practiced layout positioning and structuring components cleanly.

## 🛠️ Tech Stack
* **HTML5** - Semantic structure.
* **CSS3** - Styling and layout behavior.
* **Figma** - Design reference and asset extraction.
